# 🌿 Urban Flora

**Urban Flora** 是一个基于区块链的城市植物观察记录应用。  
它为城市中的每一株植物创建唯一的数字身份，并记录其生命周期（发芽、生长、开花、结果、落叶）。

---

## ✨ 项目背景

城市中的植物每天都在变化，但很少有人记录它们。

Urban Flora 的目标是：
> 将城市中的自然变化转化为可记录、可追溯、可分享的数字记忆。

---

## 💡 核心功能

### 🌱 1. 创建植物 NFT
- 用户可以为一株植物创建唯一的 NFT
- 记录植物名称和位置（如“公园东门”）

### 📍 2. 植物位置记录
- 使用文本或经纬度标记植物所在位置
- 避免复杂地图集成，专注核心功能

### 🌸 3. 生命周期记录
用户可以为植物添加观察记录：
- 🌱 发芽
- 🌿 生长
- 🌸 开花
- 🍎 结果
- 🍂 落叶

### 🕒 4. 成长时间线展示
- 以时间线形式展示植物的变化过程
- 构建“植物的一生”记录

## 🚀 快速开始

### 1. 部署智能合约
- 使用 Remix 编译并部署合约
- 获取合约地址和 ABI

### 2. 启动后端服务

```bash
mvn spring-boot:run
```

默认地址：
```
http://localhost:8080
```

---

### 3. 启动前端

```bash
npm run dev
```

默认地址：

```
http://localhost:8081
```
